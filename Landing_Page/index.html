Project Name: WorldSurf

Project Description

This is my project to test Interface Usability for a WorldSurf site I am working on in class. This site will display links to take the Usability test on NavFlow and also a link to the results of those tests. By using different design principles I aim to make the site easy to follow and understand. Throughout the testing phases the results I receive will help me with using the best techniques in the optimal way so that navigation on the site is fluid and effortless.
